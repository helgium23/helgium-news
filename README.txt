Jekyll minimal blog for GitHub Pages
------------------------------------
1) Buat repo publik di GitHub, upload semua file/folder dalam ZIP ini ke root repository.
2) Aktifkan GitHub Pages: Settings → Pages → Source: Deploy from a branch → branch: main → folder: /(root).
3) Pastikan fitur GitHub Pages aktif (butuh 1-2 menit untuk build pertama).
4) Semua artikel harus disimpan di folder _posts/ dengan nama file: YYYY-MM-DD-slug.md dan front matter Jekyll.
5) Workflow n8n yang saya sertakan akan otomatis membuat file markdown di folder _posts/ sesuai format ini.
